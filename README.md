# TypeScript Starter Template

Minimal template for TypeScript development with ESLint + Prettier formatting.

## Setup

```
npm install
```

## Scripts

`npm run dev` → run the project with ts-node

`npm run build` → compile TypeScript into /dist

`npm run start` → execute the compiled code

`npm run lint` → check code style with ESLint

`npm run format` → format code with Prettier

`npm test` → placeholder for tests

## Features

- TypeScript 5.x

- ESLint + Prettier configured with StandardJS-like style (single quotes, no semicolons, 2-space indentation)

- GitHub Actions for CI (lint, build, test)
